In den Visual Studio Code-Anweisungen wird die .NET Core-CLI für ASP.NET Core-Entwicklungsfunktionen wie die Projekterstellung verwendet. Sie können diesen Anweisungen auf jeder beliebigen Plattform (macOS, Linux oder Windows) und in jedem beliebigen Code-Editor folgen. Kleinere Änderungen sind möglicherweise erforderlich, wenn Sie ein anderes Programm als Visual Studio Code verwenden. Weitere Informationen zum Installieren von Visual Studio Code unter macOS finden Sie unter [Visual Studio Code unter macOS](https://code.visualstudio.com/docs/setup/mac).
